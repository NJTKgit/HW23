# HW23 ReadME
Постановка задачи: создать парсер, принимающий на вход номер пользователя, получающий на выходе список оцененных пользователем фильмов и их средней оценкой по сервису.

Инструкция по сборке и эксплуатации кода:
1. Закинуть файл Module23HM.py в папку с проектом.
2. Открыть файл Module23HM.py через PyCharm.
3. В 23 строчке (user_rates = collect_user_rates('217658719')) в скобках вписать необходимый номер пользователя.
4. Запустить проект.
5. В папке с проектом создастся файл "user_rates.xlsx", в котором будет вся информация.
